## GUIA DOMÍNIO E INFRAESTRUTURA

### Comprar domínio .pt
1. Aceder a: https://www.eurodns.com/pt ou https://www.pt.domains
2. Pesquisar "bookme.pt"
3. Custo: ~€15/ano
4. Registar com dados empresariais (NIF obrigatório para .pt)

### Alternativas se bookme.pt não disponível:
- bookme.app
- getbookme.com  
- bookme.co
- marcacoes.pt

### Configurar após compra:
```
1. Vercel Dashboard → projeto → Settings → Domains → Add
2. Inserir domínio: bookme.pt
3. Copiar registos DNS (A + CNAME) fornecidos pelo Vercel
4. Inserir no painel do registrador
5. Aguardar propagação (max 24h, geralmente 30 min)
6. Atualizar .env no Vercel: VITE_APP_URL=https://bookme.pt
7. Recriar Stripe Webhook: https://bookme.pt/api/webhooks/stripe
8. Configurar Resend: verificar domínio bookme.pt
9. Supabase: Authentication → URL Configuration → Site URL: https://bookme.pt
```
